# Soy Darks Website

## Overview
The "Soy Darks" website is a modern platform dedicated to showcasing a collective of DJs with a gothic aesthetic. The site serves as a hub for fans and followers to explore the collective's history, individual DJs, upcoming events, and a gallery of past performances.

## Project Structure
```
soy-darks-website
├── src
│   ├── index.html          # Main landing page
│   ├── about.html          # Information about the collective
│   ├── djs.html            # Profiles of individual DJs
│   ├── events.html         # Upcoming events and ticket information
│   ├── gallery.html        # Gallery of images and videos
│   ├── contact.html        # Contact form and booking inquiries
│   ├── css
│   │   ├── main.css        # Main styles
│   │   ├── gothic-theme.css # Gothic aesthetic styles
│   │   └── responsive.css   # Responsive design styles
│   ├── js
│   │   ├── main.js         # Main JavaScript functionality
│   │   ├── events.js       # Events section functionality
│   │   └── gallery.js      # Gallery functionality
│   └── assets
│       └── fonts           # Custom fonts for the website
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd soy-darks-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open the `src/index.html` file in your web browser to view the website.

## Features
- **Gothic Aesthetic**: The website incorporates a dark, gothic theme with custom fonts and styles.
- **DJ Profiles**: Each DJ in the collective has a dedicated page showcasing their biography and social media links.
- **Event Listings**: Users can view upcoming events, including details and ticket information.
- **Gallery**: A visual showcase of past events and performances.
- **Contact Form**: A section for inquiries and booking requests.
